4.1 Breakpoints

Breakpoints are sets in the map $breakpoints

  • xs: 0px,
  • sm: 576px,
  • md: 768px,
  • lg: 992px,
  • xl: 1200px

Breakpoints can be call in the sass project using the functions:

bp($bp) which will outpout a value in em

bp-var($bp, $true-val:false) which will outpout the css variable

Usage of bp():

  • bp(sm); => 36em

Usage of bp-var():

  • bp-var(sm); => var(--bp-sm); If $use-css-var = true
  • bp-var(sm); => 576px; If $use-css-var = false
  • bp-var(sm, true) => 576px;

4.3 Container Sizes

Container sizes are sets in the map $container-sizes

  • sm
  • md
  • lg
  • xl

Container Sizes can be call in the sass project using the function:

container($container-size, $true-val:false).

Function examples to call the map:

  • container(md); => var(--container-size-sm); If $use-css-var = true
  • container(md); => 720px; If $use-css-var = false
  • container(md, true) => 720px;

4.4 Font Families

Font Families are sets in the map $font-families

  • paragraph: Lato,
  • title: Rubik,
  • highlight: Rubik

Font Families can be call in the sass project using the functions:

font-family($font-family, $true-val:false) which will outpout by default the the css variable

Usage of font-family():

  • font-family(title) => var(--font-family-title) If $use-css-var = true
  • font-family(title) => "Rubik", "Helvetica Neue", Arial, sans-serif If $use-css-var = false
  • font-family(title, true) => "Rubik", "Helvetica Neue", Arial, sans-serif
