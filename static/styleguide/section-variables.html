12.2 Breakpoints

Breakpoints are sets in the map $breakpoints

  • xs: 320px, // Smartphone
  • sm: 600px, // Tablets
  • md: 900px, // Tablets Landscape and small desktops
  • lg: 1200px, // Desktops
  • xl: 1800px, // Large Desktop

Breakpoints can be call in the sass project using the functions:

bp($bp) which will output a value in em

bp-var($bp, $true-val:false) which will output the CSS variable

Usage of bp():

  • bp(sm); => 36em

Usage of bp-var():

  • bp-var(sm); => var(--bp-sm); If $use-css-var = true
  • bp-var(sm); => 576px; If $use-css-var = false
  • bp-var(sm, true) => 576px;

12.3 Colors

Colors are set in the map $color-themes. Each color themes have 5 variables:

  • Base
  • Dark
  • Light
  • Transparent
  • Contrast

Colors can be call in the sass project using the functions:

Usage of font-family():

  • color(success, light) => #76bd7a If $use-css-var = false
  • color(success, light, true) => #76bd7a

12.7 Grid

This project use by default CSS grid layout display: grid for its grid. The grid can be use a rigid with a number of set columns define by $grid-columns (here 12 columns). Or the grid can be use as fluid using the map $grid-item-sizes with $grid-item-sizes-base (here 80px) See The grid section in layout for more informations

  • xs: $grid-item-sizes-base,
  • sm: $grid-item-sizes-base * 2.5,
  • md: $grid-item-sizes-base * 4,
  • lg: $grid-item-sizes-base * 5,
  • xl: $grid-item-sizes-base * 6,
  • xxl: $grid-item-sizes-base * 8

Grid Item sizes can be call in the sass project using the functions:

grid-item-size($grid-item-size, $true-val:false) which will output by default the CSS variable

Usage of grid-item-size():

  • grid-item-size(md) => var(--grid-item-size-md) If $use-css-var = true
  • grid-item-size(md) => 320px If $use-css-var = false
  • grid-item-size(md, true) => 320px

12.8 Spacers

Spacers is use to set padding and margin in the project. The base value is set with $spacer-reference (1rem)

  • 0: 0,
  • 1: ($spacer-reference * 0.25),
  • 2: ($spacer-reference * 0.5),
  • 3: $spacer-reference,
  • 4: ($spacer-reference * 1.25),
  • 5: ($spacer-reference * 1.5),
  • 6: ($spacer-reference * 3),
  • 7: ($spacer-reference * 6),
  • 8: ($spacer-reference * 9),
  • 9: ($spacer-reference * 12)

spacer sizes can be call in the sass project using the functions:

spacer($spacer, $true-val:false) which will output by default the CSS variable

Usage of grid-item-size():

  • spacer(4) => var(--spacer-4) If $use-css-var = true
  • spacer(4) => 1.25rem If $use-css-var = false
  • spacer(4, true) => 1.25rem

12.10 z-index

z-index are sets in the map $z-indexes. The first entry will get the highest value and the last one the lowest. You can find an example in this pen

  • skip-navigation
  • main-navigation

z-index indexes can be call in the sass project using the functions:

z($name) which will output the reverse index of $name time 10

Usage of font-size():

  • z(main-navigation) => 10
