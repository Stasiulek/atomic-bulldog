10.7 Grid

This project use by default CSS grid layout display: grid for its grid. The grid can be use a rigid with a number of set columns define by $grid-columns (here 12 columns). Or the grid can be use as fluid using the map $grid-item-sizes with $grid-item-sizes-base (here 80px) See The grid section in layout for more informations

  • xs: $grid-item-sizes-base,
  • sm: $grid-item-sizes-base * 2.5,
  • md: $grid-item-sizes-base * 4,
  • lg: $grid-item-sizes-base * 5,
  • xl: $grid-item-sizes-base * 6,
  • xxl: $grid-item-sizes-base * 8

Grid Item sizes can be call in the sass project using the functions:

grid-item-size($grid-item-size, $true-val:false) which will output by default the CSS variable

Usage of grid-item-size():

  • grid-item-size(md) => var(--grid-item-size-md) If $use-css-var = true
  • grid-item-size(md) => 320px If $use-css-var = false
  • grid-item-size(md, true) => 320px
