Container Sizes

Breakpoints are sets in the map $breakpoints

  • xs: 0px,
  • sm: 576px,
  • md: 768px,
  • lg: 992px,
  • xl: 1200px

Breakpoints can be call in the sass project using the function bp($bp) which will outpout a value in em or using bp-var($bp, $true-val:false)

Functions example:

  • bp(sm); - Output 36em;
  • bp-var(sm) - If $use-css-var is set to true the output will be var(--bp-sm); else it will output the value in px 576px
  • bp-var(sm, true) Output the value in px 576px
